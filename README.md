## 🖥️ Projects

### 📸 SEIZMO — Face Swap 기반 얼굴 합성 포토부스

> **기간:** 2025.07.03 ~ 2026.12.03  
> **역할:** 백엔드 개발, 데이터 수집  
> **수상:** 성결대학교 AI활용 경진대회 3위 🥉

**문제 해결 경험**

단일 노트북 환경에서 Gemini API 사용 시 얼굴 합성에 5분 이상 소요되는 심각한 속도 저하 문제가 발생했습니다.  
**InsightFace(PyTorch)** 모델을 도입하고 **Anaconda 서버 환경을 최적화**하여 합성 시간을 **1분 이내로 90% 이상 단축**했습니다.

[📁 프로젝트 상세 보기](https://github.com/Jungyh0/SEIZMO)

### 📝 리뷰 분석 Agent — LLM 기반 사용자 리뷰 분석 및 피드백 AI Agent

> **기간:** 2026.05.11 ~ 2026.05.13  
> **역할:** DB 관리, State 정의 및 관리, 피드백 분석 Agent 설계 및 구현  
> **소속:** KT AIVLE School 미니 프로젝트

**문제 해결 경험**

Supervisor 패턴의 Agent만으로는 부정적 리뷰에 대한 피드백 기능 구현이 어려운 상황이었고, 팀 내에서 기능 축소 의견이 나왔습니다.  
직접 **Router 패턴** 기반의 피드백 분석 Agent를 설계하고 구현하여 기능을 완성했습니다.
